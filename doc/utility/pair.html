Length

number result utility.pair.length(number x, number y)

Returns the length of the vector (x, y)

Length Squared

number result utility.pair.length2(number x, number y)

Returns the squared length of the vector (x, y)

Normalize

(number x', number y') utility.pair.normalize(number x, number y)

Returns the unit vector from the given vector (x, y)
Returns (0, 0) if the vector is invalid or has zero length

Rotate

(number x', number y') utility.pair.rotate(number x, number y, number r)

Rotates the vector (x, y) around the origin and returns the resulting vector

Dot Product

number result utility.pair.dot(number x1, number y2, number x2, number y2)

Returns the dot product of the vectors (x1, y1) and (x2, y2)

Projection

(number x', number y') utility.pair.project(number x1, number y1, number x2, number y2)

Returns the vector projection from (x1, y1) onto (x2, y2)
