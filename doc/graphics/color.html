ID: graphics.color


version 1.0

Status:


Lists named colors and provides color manipulation functions.

Get Color

table result, boolean success graphics.color.get(string name)

Returns the color with the name `name`.

Set Color

boolean success graphics.color.set(string name)

Sets the color for use in drawing.
Equivalent to love.graphics.setColor(color:get(name)), but with protection.
Addtionally, issues a warning on failure.

Add Color

void graphics.color.add(string name, number r, number g, number b, number a)

Adds a color of name `name` and components (r, g, b, a)

Make Static Color

table color graphics.color.rgb_static([number r, number g, number b, number a])

Creates an immutable, static color object; used in system colors.

Make Color from RGB

table color graphics.color.rgb([number r, number g, number b, number a])

Creates an RGB color with components (r, g, b, a) and returns it.
`r`, `g`, and `b` default to 0, while `a` defaults to 255.

Make Color from HSV

table color graphics.color.hsv([number h, number s, number v, number a])

Creates an RGB color with the HSV components (h, s, v, a) and returns it.

Make Color from HSL

table color graphics.color.hsl([number h, number s, number l, number a])

Creates an RGB color with the HSL components (h, s, l, a) and returns it.

Color Mix

color target graphics.color.mix(number ratio, color first, color second, [color target])

Mixes colors `first` and `second`, putting the result in `target`.
By default, `target` is a new color object.
